# Work Day Scheduler

## Description

A simple calender application that allows a user to save events to localStorage for each hour of the work day. This app runs in the browser.

## Technologies Used

* HTML
* CSS
* JavaScript
* Bootstrap
* jQuery
* Momentjs
* FontAwesome
